              Computer Usage

              The software ag-ess-stu5 used for data analysis is located on a CD that accompanies the class textbook 'Introduction to Applied Geophysics'. On this CD there is a total of five programs available:

              • REFRACT for refraction seismology,

                      • Problems: 1 / 2 / 3 / 4 

              • REFLECT for reflection seismology,

                      • Problems: 1 / 2 / 3 / 4 / 5 / 6 / 7 / 8

              • RESIST for electrical resistivity,

                      • Problems: 1 / 2 / 3 / 4

              • GRAVMAG for gravity and magnetics, and

              • DIFFRACT for ground-penetrating radar.


              All of the software modules are designed to run on both Macintosh and Windows systems, can perform forward and inverse modeling, and support all the major exploration methods covered in the book.
